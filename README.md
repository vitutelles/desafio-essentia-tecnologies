## Tecnologias
- Angular
- Node.js
- TypeScript
- MySQL

- JWT (Auth)

## Como rodar o backend
```bash
cd backend
npm install
npm exec prisma migrate dev
npm run dev
```

## Como rodar o frontend
```bash
cd frontend
npm install
npm start
```
